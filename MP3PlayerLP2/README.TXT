Para que o programa seja executado, se encontram necessários alguns pré-requisitos.
Pré-requisitos esses que são programas e arquivos para executar código Java. Inicialmente, o
usuário deve configurar um ambiente de desenvolvimento de sua preferência qualquer. Além
disso, é necessário ter o JRE (Java Runtime Environment) instalado para rodar o código-fonte,
juntamente do JavaFX para a interface gráfica do programa.
